## Usage

```bash
# create an app
$ nuxtron init my-app --example with-javascript

# install dependencies
$ cd my-app
$ yarn (or `npm install`)

# development mode
$ yarn dev (or `npm run dev`)

# production build
$ yarn build (or `npm run build`)
```
